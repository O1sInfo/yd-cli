# ydapi

Youdao Fanyi Command-Line Interface Using Youdao Api.

# Usage

1. `git clone https://github.com/O1sInfo/ydapi.git` or Download as zip.
2. (Optional) Put **yd.bat** in your envoiroment path.
3. `yd -h` for HELP. 

# Example

* `yd -w word`
```
n        [语] 单词；话语；消息；诺言；命令,  (Word)人名；(英)沃德
vt       用言辞表达
```
* `yd -t `
